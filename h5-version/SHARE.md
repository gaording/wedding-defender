# 📱 春节分享完整指南

## 第一步：部署（3选1）

### ⭐ 推荐：Vercel（免费、快速）

```
1. 打开 https://vercel.com
2. 用GitHub登录（推荐）
3. 点击 "New Project"
4. 拖拽 h5-version 文件夹到页面
5. 等待30秒
6. 获得链接：https://wedding-defender-xxx.vercel.app
```

### 最简单：Netlify Drop

```
1. 打开 https://app.netlify.com/drop
2. 拖拽 h5-version 文件夹
3. 几秒后获得链接
```

### 技术流：GitHub Pages

```
1. 创建GitHub仓库
2. 上传 h5-version 所有文件
3. Settings → Pages → 启用
```

## 第二步：生成二维码

### 方法1：用脚本生成

```bash
cd h5-version
./generate-qrcode.sh "https://你的链接"
```

### 方法2：在线生成

访问任意网站：
- https://cli.im/ （草料二维码）
- https://www.qrcode-generator.com/
- https://api.qrserver.com/

输入你的链接即可下载

### 方法3：在线工具（推荐）

**草料二维码：** https://cli.im/
1. 选择"网址"
2. 输入你的链接
3. 点击"生成二维码"
4. 下载图片

## 📊 跟踪数据（可选）

### 想知道多少人用了？

**方案1：百度统计（免费）**
1. 访问 https://tongji.baidu.com/
2. 注册并添加网站
3. 复制统计代码
4. 粘贴到 `index.html` 的 `<head>` 中

**方案2：简单计数**
```html
<!-- 在index.html中添加 -->
<script>
// 简单访问计数
let visits = localStorage.getItem('visits') || 0;
visits++;
localStorage.setItem('visits', visits);
console.log('这是第' + visits + '次访问');
</script>
```

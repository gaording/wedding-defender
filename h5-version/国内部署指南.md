# 🇨🇳 国内快速部署指南

## 为什么移动端打不开？

**原因：** Vercel/Netlify是国外服务，移动网络（4G/5G）访问受限

**解决：** 使用国内部署平台

---

## 🚀 方案1：Gitee Pages（推荐）

### 步骤：

1. **注册Gitee**
   - 打开 https://gitee.com/
   - 注册/登录

2. **创建仓库**
   - 点击右上角 "+" → "新建仓库"
   - 仓库名：`wedding-defender`
   - 选择"公开"
   - 点击"创建"

3. **上传文件**

   方式A：网页上传
   ```
   进入仓库 → 文件 → 上传文件
   把 h5-version 里所有文件上传
   ```

   方式B：Git推送（如果有Git基础）
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://gitee.com/你的用户名/wedding-defender.git
   git push -u origin master
   ```

4. **启用Gitee Pages**

   ```
   仓库 → 服务 → Gitee Pages
   → 启动服务
   → 等待部署
   → 获得：https://你的用户名.gitee.io/wedding-defender/
   ```

5. **完成！**
   - 手机、微信都能访问
   - 速度还快！

---

## 🚀 方案2：Coding Pages

### 步骤：

1. **注册Coding**
   - 打开 https://coding.net/
   - 注册/登录

2. **创建项目**
   - 新建项目 → 静态网站
   - 上传 h5-version 文件

3. **开启Pages**
   - 项目 → 持续部署 → 静态网站
   - 立即部署
   - 获得：https://你的用户名.coding.net/pages/项目名

---

## 🚀 方案3：阿里云OSS（需要花钱，但最稳定）

### 步骤：

1. **开通OSS**
   - 阿里云控制台 → 对象存储OSS
   - 创建Bucket
   - 设置为"公共读"

2. **上传文件**
   - 上传 h5-version 所有文件
   - 设置静态网站托管

3. **绑定域名**（可选）
   - 如果有域名，可以绑定

---

## 🚀 方案4：自己服务器（有服务器的话）

```bash
# 使用Python临时服务器
cd h5-version
python3 -m http.server 8000

# 或使用Node.js的http-server
npx http-server -p 8000

# 然后用内网穿透（如ngrok）
npx ngrok http 8000
```

## ⚠️ 常见问题

**Q: Gitee Pages访问慢？**
A: 第一次访问可能慢，后续会快

**Q: 微信内打不开？**
A: 可能需要在微信公众平台配置域名白名单（需要备案域名）

**Q: 想用自定义域名？**
A: 需要备案，完成后可以绑定到Gitee Pages

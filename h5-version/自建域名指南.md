## 🎯 方案1：Gitee Pages（推荐）

### 优点
- ✅ 免费
- ✅ 国内速度快
- ✅ 微信内可访问
- ✅ 5分钟部署完成

### 步骤

#### 1. 注册Gitee
```
https://gitee.com/
手机号注册，无需实名
```

#### 2. 创建仓库
```
右上角 "+" → 新建仓库
仓库名：wedding-defender
选择：公开仓库
初始化：勾选
```

#### 3. 上传文件
```
方式A：网页上传
- 进入仓库
- 文件 → 上传文件
- 选择 h5-version 所有文件
- 提交

方式B：Git命令
cd h5-version
git init
git add .
git commit -m "Initial"
git remote add origin https://gitee.com/你的用户名/wedding-defender.git
git push -u origin master
```

#### 4. 启用Gitee Pages
```
仓库页面 → 服务 → Gitee Pages
部署来源：master目录
点击：启动
等待1-2分钟
访问：https://你的用户名.gitee.io/wedding-defender/
```

#### 5. 测试微信访问
```
把链接发到微信
手机点击打开
✅ 正常访问
```

## 🌐 方案2：香港服务器（无需备案）

### 优点
- ✅ 无需备案
- ✅ 自有域名
- ✅ 微信可访问
- ✅ 完全控制权

### 缺点
- ❌ 需要花钱（50-100元/月）
- ❌ 需要一定技术能力

### 推荐服务商

| 服务商 | 香港节点 | 价格 |
|--------|----------|------|
| Vultr | HK | $5/月起 |
| DigitalOcean | Singapore | $6/月起 |
| 阿里云 | HK | ¥24/月起 |
| 腾讯云 | HK | ¥20/月起 |

### 详细步骤

#### 1. 购买服务器
```bash
# 推荐 Vultr
https://www.vultr.com/

# 选择
- 地区：Hong Kong
- 系统：Ubuntu 20.04/22.04
- 配置：$5/月（512MB或1GB）
- 点击Deploy

# 等待1-2分钟
# 获得IP地址
```

#### 2. 购买域名
```bash
# 推荐（阿里云）
https://wanwang.aliyun.com/

搜索你的域名，例如：chunyuan-helper.com
.com 约 50-70元/年
.cn 约 30元/年（需要实名）

# 推荐（腾讯云）
https://dnspod.cn/

# 推荐（国外）
https://www.namecheap.com/
# 不需要实名，但国内可能被屏蔽
```

#### 3. 域名解析
```bash
# 在域名控制台添加A记录
类型：A
主机记录：@
记录值：你的服务器IP
TTL：600

# 等待10分钟生效
```

#### 4. 服务器部署
```bash
# SSH登录服务器
ssh root@你的服务器IP

# 安装Nginx
apt update
apt install nginx -y

# 上传文件
# 方式A：使用scp
scp -r h5-version/* root@IP:/var/www/html/

# 方式B：在服务器上直接创建
cd /var/www/html/
rm index.nginx-debian.html
# 用vi或nano创建index.html
# 把h5-version/index.html内容复制进去

# 启动Nginx
systemctl start nginx
systemctl enable nginx
```

#### 5. 访问测试
```bash
# 浏览器访问
http://你的域名.com

# 成功！
```

#### 6. 配置HTTPS（可选）
```bash
# 使用Let's Encrypt免费证书
apt install certbot python3-certbot-nginx -y

certbot --nginx -d 你的域名.com

# 自动续期
certbot renew --dry-run

# 完成！
https://你的域名.com
```

## 🔧 方案3：Cloudflare Pages（免费CDN）

### 优点
- ✅ 免费
- ✅ 全球CDN
- ✅ 自动HTTPS
- ⚠️ 微信访问**可能**被拦截

### 步骤

#### 1. 注册Cloudflare
```
https://dash.cloudflare.com/
邮箱注册
```

#### 2. 添加域名
```
Add a Site
输入你的域名
选择Free计划
```

#### 3. 修改DNS
```
在你的域名注册商
把NS服务器改为Cloudflare提供的两个
- ada.ns.cloudflare.com
- bob.ns.cloudflare.com

等待生效（10分钟-24小时）
```

#### 4. 部署到Cloudflare Pages
```
Pages → Create a project
选择：Connect to Git
连接：GitHub或Gitee
选择：wedding-defender仓库
构建命令：留空
输出目录：/

开始部署
```

#### 5. 绑定自定义域名
```
Pages项目 → Custom domains
添加：你的域名.com

Cloudflare自动配置DNS
```

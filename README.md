# 催婚防御助手 - 微信小程序

## 项目简介

帮助大龄单身青年优雅应对春节催婚问题的微信小程序。

> 你不是一个人在战斗！让我们用幽默、智慧和优雅，化解每一次尴尬的催婚对话。

## 功能模块

### 1. 首页
- 每日一句/安慰 - 每天温暖的鼓励
- 快速入口 - 一键直达核心功能
- 今日幸运buff - 游戏化运势系统
- 数据统计 - 展示你的应对成果

### 2. 武器库（核心功能）
- **场景化话术生成器** - 根据提问对象、问题类型、回应风格生成话术
- **表情包库** - 应对催婚的表情包合集
- **数据反击卡** - 用数据和事实说话

### 3. 树洞社区
- 匿名吐槽 - 释放情绪
- 求助帖 - 获得建议
- 经验分享 - 帮助他人
- 心情记录 - 见证成长

### 4. 沟通桥梁
- 给父母的一封信 - 表达想法的模板
- 沟通指南 - 学习有效沟通
- 心理自测 - 了解自己
- 应对技巧 - 实用策略

### 5. 剧本杀游戏
- 游戏化练习催婚应对
- 优雅度、气氛值、杀伤力三维评分
- 解锁成就称号

### 6. 我的
- 催婚日记 - 记录每次应对
- 成就系统 - 游戏化激励
- 称号展示 - 荣誉时刻
- 数据统计 - 看见自己的进步

## 快速开始

### 环境要求
- 微信开发者工具（最新版）

### 安装步骤

1. **克隆/下载项目**
   ```bash
   cd wedding-defender
   ```

2. **准备图标文件**
   - 在 `images/` 目录下放置以下图标（建议 81x81 px）：
     - `tab-home.png` / `tab-home-active.png`
     - `tab-arsenal.png` / `tab-arsenal-active.png`
     - `tab-community.png` / `tab-community-active.png`
     - `tab-profile.png` / `tab-profile-active.png`
   - 可从 [iconfont](https://www.iconfont.cn/) 或 [iconpark](https://iconpark.oceanengine.com/) 下载

3. **打开微信开发者工具**
   - 导入项目目录
   - 填写 AppID（测试可用"测试号"）
   - 点击"编译"运行

### 配置说明

在 `project.json` 中修改 `appid` 为你自己的小程序 AppID：
```json
{
  "appid": "your_appid_here"
}
```

## 核心功能说明

### 话术生成器

支持以下维度组合：

**提问对象：**
- 👨‍👩‍👧 父母
- 👵 亲戚
- 👥 同辈
- 👴 长辈

**问题类型：**
- 💍 什么时候结婚
- 💑 有对象了吗
- 🏠 什么时候带回来
- 👶 别人家孩子...

**回应风格：**
- 🌸 温和委婉
- 😄 幽默自嘲
- 🧠 理性分析
- ↩️ 反客为主
- 📖 哲学金句

### 成就系统

用户可通过以下行为获得成就：
- 成功应对催婚问题
- 书写催婚日记
- 社区互动发帖
- 完成剧本杀游戏

**称号示例：**
- 🏆 太极拳宗师 - 成功应对10次
- 🏆 坚守者 - 成功应对30次
- 🏆 记录者 - 写满5篇日记
- 🏆 社区活跃者 - 发3个帖子
- 🏆 外交官 - 游戏中高优雅高气氛

## 数据存储

当前版本使用本地存储，数据保存于用户设备：
- `userInfo` - 用户信息
- `achievements` - 成就数据
- `titles` - 称号列表
- `favoriteScripts` - 收藏的话术
- `diaries` - 日记记录
- `communityPosts` - 社区帖子

**注意：** 生产环境建议接入后端服务或使用微信云开发。

   - 小程序介绍

   - 微信开发者工具 → 上传
